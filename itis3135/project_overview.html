Dynamic Functionality of the Website

  1. Poll of the Week
    • Presents users with a fun Marvel-related poll question each week. Users will be able to click on an option and instantly see which choice is more popular. The poll will be embedded from Google Forms and some js for more interactivity and transitions.
    • Page: index.html
    • Why: It would encourage weekly return visits and member engagement through fun, esay interactivity
    • Example:
      • Strawpoll
  2. Interactive Photo Gallery
    • Showcases photos from past events and themed activities. Users can click images to open a larger version in a lightbox, along with scrolling through photo albums
    • Page: gallery.html
    • Why: It makes the gallery more user-friendly and visually engaging without cluttering the page with too much at once.
    • Examples:
      1. W3S How to CSS Image Gallery Scroll
      2. Lightbox2
